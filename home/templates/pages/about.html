About This Website

In this page I'll briefly describe the process of creating this Website, in case you are interested.

The Concept

The idea of this Website is to be a portfolio of my work. In here, I'll introduce myself on the Homepage and display all my personal projects.

The reason why I use this Website as well and not only LinkedIn/Github is because this way I can deploy my Machine Learning Models in a Production Environment and put it to work in a way Github alone wouldn't let me.
For example, check my fully functional Rent Price Calculator by clicking here.

Not only that, but this way I can demonstrate a little bit of my Web Development skills. A skill not always used in Data Science, but one that can come in handy in specific situations, such as in the calculator mentioned above.

The Execution

In order to create this site, I used the Python Web Framework Django. This Framework lets you create a Website fairly simply. You can develop both Front-End and Back-End inside Django using Pure Python. Although it is also possible to use other Frameworks together with Django such as React (not used in this particular project).

For more info about Django, check:
https://www.djangoproject.com

Not only that, but for this project, I was really lucky to find the Pixel Bootstrap created by AppSeed. This Open Source Bootstrap gives you all the basic elements to create a Website. Thanks to them, I didn't have to create any of the CSS or Javascript you see in here. Everything you see here I used their strong foundation to create.

For more info about AppSeed, check:
https://appseed.us

The Full Website Project

Even tough AppSeed made my life much easier, there's still plenty of customization I had to do to make this Website looks as you see right now.
If you are curious to know exactly how I used their Bootstrap and all the steps I took to make this Website how it is at this moment, you can check the full project on my GitHub, by clicking the link below:
https://github.com/danilorr/django-pixel
